# Early-life_GPI
Unveiling the unknown, latent and inter-generational effect of early-life Gastrointestinal Parasitic infections (GPI) in unmanipulated wild hosts. 
This repository contains the source code and data for the comprehensive assessment of these crucial fitness gaps of the paper "Compromised futures: gastrointestinal parasites reduce survival and recruitment but not reproductive performance in wild female spotted hyenas (Crocuta crocuta)".

Note: ID were removed and are random numbers.
